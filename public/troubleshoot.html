📱 ngrok Free Tier Notice

Why only Chrome Incognito works:

ngrok free tier shows a warning page on first visit. Some browsers cache this warning page aggressively.

Solutions:

  1. Use Chrome or Edge (best compatibility)
  2. Clear browser cache completely
  3. Use Incognito/Private mode
  4. Click "Visit Site" on ngrok warning page
