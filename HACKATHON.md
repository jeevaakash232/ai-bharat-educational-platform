# 🚀 AI Bharat — Hackathon Submission

> *AI-powered education for every Indian student, in their own language.*

---

## 🎯 Problem Statement

Millions of students across India face a broken education system:

- One-size-fits-all content ignores regional languages and state boards
- No access to personalized tutoring outside major cities
- Students who learn in Tamil, Hindi, Telugu, or other regional languages are forced to study in English
- Zero real-time feedback on performance or at-risk subjects

**The result:** talented students fall behind — not because they lack ability, but because the system doesn't speak their language.

---

## 💡 Solution — AI Bharat

A full-stack AI-powered learning platform built for **Classes 1–12**, personalized to each student's **state, board, medium, and mother tongue**.

Every student gets a learning experience tailored to who they are — not a generic one built for someone else.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🧠 AI Assistant | AWS Bedrock (Claude) answers doubts in Tamil, Hindi, Telugu & 10+ Indian languages |
| 📝 Regional Language Quizzes | AI-generated quizzes in the student's own language, aligned to their class & board |
| 📊 Performance Predictions | AI predicts exam scores, detects at-risk subjects, and simulates study improvements |
| 🎥 Recorded Classes | Teachers upload class recordings; students watch anytime |
| 🌐 Multilingual Chat | Auto language detection — type in any Indian language, get answers in the same language |
| 🏃 Discipline Tracker | Streak tracking, focus monitoring, delayed gratification training |
| 💼 Career Simulator | Simulates real-world career paths based on current study habits |
| ❤️ Health Tracker | Monitors sleep, screen time, and study hours for student wellness |

---

## 🛠️ Technologies Used

**Frontend**
- React 18 + Vite
- Tailwind CSS
- Lucide React icons
- Firebase Authentication (Google Sign-In)

**Backend**
- Node.js + Express
- AWS Bedrock (Claude — `google.gemma-3-12b-it`)
- AWS Translate — multilingual response translation
- AWS Comprehend — automatic language detection
- AWS S3 — book and video storage
- AWS DynamoDB — user history and predictions

**AI / ML**
- Python microservices for study twin predictions
- Focus meter AI
- Teacher assistant content generation

**Deployment**
- Frontend: Vercel
- Backend: Render
- Database: DynamoDB + S3

---

## 🌍 Impact & Real-World Use Case

- A Class 10 student in Tamil Nadu can ask doubts in Tamil and get step-by-step answers in Tamil
- A teacher in rural Maharashtra can upload recorded classes for students with no live internet
- A student in Bihar studying in Hindi medium gets quizzes generated entirely in Hindi script
- Parents can monitor their child's predicted exam score and at-risk subjects in real time

**Target users:** 250+ million school students across India, spanning 28 states and 10+ regional languages.

---

## 🏗️ Build Journey

Built end-to-end in a hackathon sprint — from a blank repo to a fully deployed, multilingual AI education platform. Every architectural decision was made with one constraint: *it must work for a student in a village, not just a city.*

The hardest problem wasn't the AI — it was making the AI speak the right language at the right time, for the right student.
